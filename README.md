# DePaul Dataset - Week 1 EDA

## Overview
This project contains an exploratory data analysis (EDA) of the DePaul Dataset completed as part of the Excelerate Data Visualization Trainee Internship.

## Dataset Summary
- Total Records: 7,543
- Total Variables: 80
- Unique Applicants: 5,000
- Countries Represented: 95

## Analysis Performed
- Dataset Familiarization
- Data Quality Assessment
- Missing Value Analysis
- Duplicate Analysis
- Country Analysis
- Program/Intake Analysis
- College Preference Analysis
- Application Source Analysis

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- VS Code

## Key Findings
- India was the largest applicant source country.
- Technology and business programs showed the highest demand.
- Jarvis College and Kellstadt Graduate School of Business accounted for over 84% of first-choice selections.
